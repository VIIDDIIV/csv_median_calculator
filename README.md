# csv_median_calculator

Консольное приложение для расчёта инкрементальной медианы цен из CSV-файлов с биржевыми данными.

## Описание

Приложение читает CSV-файлы из указанной директории, фильтрует их по маскам, объединяет данные из всех подходящих файлов, сортирует по времени получения (`receive_ts`)
и инкрементально вычисляет медиану цены. Каждое изменение медианы (с точностью до 8 знаков после запятой) фиксируется в выходном файле `median_result.csv` вместе с временной меткой.

### Ключевые особенности
- Поддержка файлов форматов `level.csv` (6 полей) и `trade.csv` (5 полей).
- Полная валидация CSV-файлов: при любой ошибке (неверный заголовок, недостаточно колонок, некорректные числа) файл исключается целиком.
- Автоматический выбор режима обработки в зависимости от объёма данных:
  - **in-memory** – для данных, помещающихся в оперативную память (быстрая сортировка всего массива).
  - **потоковый** – для данных, которые не умещаются в вектор, но кучи для медианы ещё влезают (многопутевое слияние с приоритетной очередью).
- Многопоточное чтение файлов для ускорения ввода-вывода.
- Буферизированная запись результатов.
- Логирование через `spdlog` с цветным выводом в консоль.
- Измерение времени выполнения ключевых этапов (вставка в кучи, операции с очередью).

## Требования

- Компилятор с поддержкой C++17 (рекомендуется C++20/23).
- CMake версии 3.23 или выше.
- Библиотеки (автоматически загружаются через `FetchContent`):
  - Boost.Program_options
  - toml++
  - spdlog

## Сборка проекта

```bash
# Клонируйте репозиторий (или распакуйте архив)
cd csv_median_calculator

# Создайте директорию сборки и выполните конфигурацию
cmake -B build -S . -DCMAKE_BUILD_TYPE=Release

# Соберите проект
cmake --build build --config Release
